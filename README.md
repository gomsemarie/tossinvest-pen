# 토스 차트 펜 (tossinvest-pen)

토스증권(tossinvest.com) 웹 차트 위에 **자유롭게 그림을 그릴 수 있는** 크롬 확장(MV3)입니다.
그린 선은 차트의 **가격·시간에 앵커링**되어 줌/스크롤 시 함께 움직이고, **종목·시간대별로 자동 저장·복원**됩니다.

> TradingView 의 도형 API 를 쓰지 않고, 차트 좌표계만 빌려 **자체 캔버스에 직접 렌더**합니다.
> 덕분에 봉 단위 스냅 없이 그린 그대로 표현되고, 마지막 봉 너머(미래 영역)에도 제약 없이 그릴 수 있습니다.

## 주요 기능

- ✏️ **펜** — 좌클릭 드래그로 자유 그리기 (색상 5종 / 굵기 3단)
- 🧽 **지우개** — 드래그로 지나간 부분만 지우는 부분 지우개 (우클릭으로도 지우기)
- 👁 **숨기기/보이기** — 그린 선을 지우지 않고 표시만 토글 (숨김 중엔 차트 자유롭게 조작)
- ↶ **되돌리기 / 다시실행** — `Cmd/Ctrl+Z`, `Cmd/Ctrl+Shift+Z`(또는 `Ctrl+Y`)
- 🗑 **삭제** — 현재 시간대만 / 이 종목의 모든 시간대
- 📷 **캡처** — 차트 + 그림을 합쳐 PNG 로 **클립보드 복사** (`Cmd/Ctrl+V` 로 붙여넣기)
- **앵커링** — 줌/스크롤/크게보기에서도 선이 차트에 붙어 움직임
- **자동 저장** — 종목 + 시간대(1분~년)별로 따로 저장·복원
- **단축키** — `Alt+D` 펜 켜기/끄기 · `Cmd/Ctrl+Z` 되돌리기 · `Cmd/Ctrl+Shift+Z`(또는 `Ctrl+Y`) 다시실행 *(펜/지우개 모드일 때)*

## 설치 (압축해제된 확장 로드)

1. 이 저장소를 내려받기 (`Code → Download ZIP` 또는 `git clone`)
2. 크롬 주소창에 `chrome://extensions` 입력
3. 우측 상단 **개발자 모드** 켜기
4. **압축해제된 확장 프로그램 로드** 클릭 → 이 폴더(`manifest.json` 이 있는 폴더) 선택
5. `https://www.tossinvest.com/stocks/...` 종목 페이지로 이동 → 네비게이션 바에 툴바가 나타납니다

## 사용법

- 네비바(또는 크게보기 시 화면 우측 상단)의 툴바에서 **✏️ 펜**을 켜고 차트 위에 그립니다.
- **지우개**는 드래그로 문지른 부분만 지워지며, 펜/지우개 모드에서 **우클릭**으로도 지울 수 있습니다.
- **📷 캡처**를 누르면 차트와 그림이 합쳐진 이미지가 클립보드에 복사됩니다.
- 시간대(1분·일·주·년 등)를 바꾸면 해당 시간대에 그린 그림만 표시됩니다.
- 크게보기에서는 툴바가 떠서 표시되며, `⠿` 손잡이로 위치를 옮길 수 있습니다.

## 동작 원리

- 토스 차트는 **TradingView Advanced Charts** 엔진을 `<iframe>` 으로 띄웁니다 (same-origin).
- 확장은 두 역할로 분기합니다.
  - **엔진(iframe)**: 펜 입력을 캡처해 각 점을 데이터 좌표 `{시간, 가격}` 으로 저장하고, 매 프레임 현재 차트 투영(logical index → x, 가격 → y)으로 재투영해 자체 캔버스에 다시 그립니다 → 줌/스크롤에 붙어 움직이는 앵커링.
  - **툴바(최상위 프레임)**: Shadow DOM 툴바, 종목·시간대별 `localStorage` 저장/복원, 프레임 간 `postMessage` 통신.
- 캡처는 차트 캔버스들과 그림 오버레이를 합성해 PNG 로 만들어 클립보드에 복사합니다.

## 알려진 한계

- 차트 위에 직접 그리는 오버레이라 **TradingView 네이티브 도형이 아닙니다** (네이티브 도구로 선택/편집 불가, 토스 계정에 동기화되지 않음).
- 세로축이 **로그 스케일**일 때는 가격→픽셀 변환을 샘플링으로 보정하지만 미세 오차가 있을 수 있습니다.
- 그림은 시간대별로 독립 저장되며, 시간대 사이로 자동 변환되지는 않습니다.
- 토스가 차트 라이브러리(TradingView)를 업데이트하면 내부 API 변경으로 동작이 깨질 수 있습니다.
